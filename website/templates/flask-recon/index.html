Flask Recon

Welcome to Flask Recon! This is a project that I have been working on to monitor my website for unauthorized scans, attacks, and other malicious activity. I have been logging requests to my website and analyzing them to determine if they are legitimate or not. I have also been monitoring IP addresses that have been flagged as suspicious to see if they are associated with known malicious activity.

Feel free to use the tools on this website to search for IP addresses, view request logs, and more. If you have any questions or feedback, please don't hesitate to contact me.
